    무통장 입금안내

    • 먼저 고객님께서 입금할 은행을 선택합니다.
    • 입금하실 은행을 선택하실때 타행일 경우 입금 수수료가 발생합니다.
    • 입금자명은 반드시 구매자 본인 명의이어야 하므로 수정이 불가능한 상태로 표시됩니다. 예금주명에는 회사명인 (주)11번가이 표시되며 판매자가 현금영수증을 제공한다고 했을 경우 수령여부를 선택하실 수 있습니다.
    • 현금영수증은 구매확정 2일 이후에 나의 11번가 > 증빙서류발급 페이지에서 직접 출력하실 수 있습니다.
    • 해당 계좌로 5영업일 내로 입금하지 않으시면 구매거부 처리되며 계좌이체 방식으로 입금이 가능하니 직접 은행을 방문하시거나 인터넷뱅킹, 전화뱅킹, ATM기를 이용한 계좌이체를 이용하시어 입금하시면 바로 입금처리되어 결제가 완료됩니다.
    • 판매자가 취소/반품 신청하는 경우 자동으로 캐시 환불 처리됩니다.
    • 200만원 이상 결제 후 취소/반품 시 계좌 환불만 가능합니다. 환불 계좌는 나의 11번가에서 등록하실 수 있습니다.
